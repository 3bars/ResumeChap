# ResumeChap

**A browser plugin that assists you with filling out job applications more easily.**

ResumeChap is a browser extension designed to take the pain out of repetitive
job application forms. It securely stores your resume details, work history,
and common application answers, then intelligently auto-fills application fields
across popular job boards and company career portals — saving you time and
reducing errors.

## Features

- **Smart auto-fill** — Detects common job application fields (name, contact
  info, work experience, education, etc.) and fills them in with a single click.
- **Profile management** — Store one or more application profiles for different
  roles or industries.
- **Resume-aware suggestions** — Pulls relevant details from your saved resume
  to answer application questions.
- **Cross-site support** — Works across major job boards and ATS platforms.
- **Privacy-first** — Your data stays in your browser; nothing is shared without
  your consent.

## Getting Started

> 🚧 This project is in early development. Setup and installation instructions
> will be added as the plugin matures.

### Prerequisites

- A Chromium-based browser (Chrome, Edge, Brave) or Firefox.

### Installation (development)

1. Clone this repository:
   ```bash
   git clone https://github.com/3bars/ResumeChap.git
   cd ResumeChap
   ```
2. Load the extension in your browser's developer/unpacked extension mode.
